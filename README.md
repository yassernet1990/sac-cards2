# SAC Cards 2

Interactive SAC Logistics identity gallery using ten images hosted on Google Drive.

## Live site

https://yassernet1990.github.io/sac-cards2/

## Google Drive integration

The gallery loads each theme image directly from its public Google Drive file ID. The source-folder button opens the shared Drive folder.

All Drive files and the folder must be shared as **Anyone with the link — Viewer**.

## GitHub Pages

Publish from the `main` branch and the repository root in **Settings → Pages**.
